## 直达文件夹

在桌面点击 Find(访达)，快捷键

```
command+shift+g
```

会弹出一个窗口，在这个窗口中输入文件夹名称就可以快速进入了

<img src="./1.jpg" />

## 拷贝当前文件夹路径

```
command+alt+c
```

## 剪切粘贴

```
command+alt+v
```

## 快速打开 Finder

```
commond+n
```

## 快速显示隐藏文件夹
```
commond+shift+.
```

## 获取npm全局安装目录
npm root -g

## 获取npm命令程序目录
npm bin -g
