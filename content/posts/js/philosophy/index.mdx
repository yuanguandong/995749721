## 基本哲学 —— ☯️

> 道生一, 一生二, 二生三, 三生万物                                                        
> 道可道，非常道
 
javascript 中万物归一思想和中国古代几千年的道家思想有不谋而合之处，赞叹古人的这种哲学归纳思想

在多年的编程生涯中也隐约归纳了下，多多指教

|无极|太极|两仪|四象|
|----|----|----|----|
|null|undefined|bolean|string、number、function、(object \| symbol)|

> 无极生太极，太极生两仪，两仪生四象，四象生八卦，八卦生万物


计算机中的二进制，1和0是不是符合阴阳之说

`1和0 <=> true和false <=>阴与阳`

无极生太极，是不是联想到了对象构造函数、原型链……

“卦” 是不是联想到了设计模式 

function 和 object 和 卦 等是不是构建出了我们的软件世界……



## 设计哲学 —— 设计模式

javascript 语法是基本拳脚，而设计模式是一种套路，

一种功法，相当于是一种武功，蛤蟆功，狮子吼，螳螂拳，金钟罩铁布衫，降龙十八掌……

所以说会点基本拳脚不成套路不成功法，

就相当于不懂设计模式一样，师出无门。

这篇<a href="/js-design-pattern" target="_blank">方法论</a>，论的就是这些套路，这些招数——javascript设计模式
