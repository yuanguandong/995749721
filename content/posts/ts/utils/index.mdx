### 使属性变为可选

```ts
type PartialByKeys<T, K extends keyof T> = {
  [P in K]?: T[P];
} & Pick<T, Exclude<keyof T, K>>;
```

### 使属性变为必填

```ts
type RequiredByKeys<T, K extends keyof T> = {
  [P in K]-?: T[P];
} & Omit<T, K>;
```

### 判断类型是否是另一个类型的子类型

```ts
type SubtypeOf<T,U> = T extends U ? true : false
```
