## 快捷键

| 名称         | 快捷键   |
| ------------ | -------- |
| 倒角         | ctrl+b   |
| 挤出         | ctrl + e |
| 游标回到原点 | shift+c  |
| 分段         | ctrl+r   |
| 辅助线       | b        |
| 画笔         | d        |
| 拆分顶点     | v        |
| 复制         | shift+d  |
| 内插面       | i        |
| 切换渲染     | z        |

## 技巧

| 目的   | 命令               |
| ------ | ------------------ |
| 变平滑 | 平滑着色、平直着色 |
| 桥接   | 桥接循环边         |
| 分离   | 分开选中项         |
